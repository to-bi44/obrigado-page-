faca

"Faca" uma palavra em portugus que significa knife em ingls, ou seja, um instrumento cortante composto por uma lmina afiada e um cabo.

uma das ferramentas mais antigas da humanidade, usada desde a Idade da Pedra para cortar, preparar alimentos, caar e diversas outras funes.

COMO HOSPEDAR ESSA PAGINA ?

Ei! Vamos colocar essa pgina de "obrigado" fofa no ar de graa e super rpido! Como s um arquivo HTML esttico (sem banco de dados ou backend), as melhores opes em 2025 so plataformas gratuitas feitas exatamente pra isso. Vou te explicar as mais fceis, passo a passo.

1. A mais simples de todas: Tiiny Host (recomendo pra comear!)

  • literalmente arrastar e soltar o arquivo leva menos de 1 minuto!
  • Passos:
    1. Salve o cdigo que te passei como index.html no seu computador.
    2. V para: https://tiiny.host/
    3. Clique em "Upload ZIP" ou arraste o arquivo diretamente (pode zipar se quiser adicionar imagens).
    4. D um nome pro seu site (ex: obrigado-autismo).
    5. Pronto! Ele gera um link tipo seunome.tiiny.site na hora.
  • Vantagens: Grtis, sem cadastro complicado, domnio simples e rpido.
  • Perfeito pra uma pgina nica como essa!

2. Netlify (minha segunda favorita profissional e com domnio custom grtis)

  • Passos rpidos (sem GitHub obrigatrio):
    1. V para: https://app.netlify.com/drop
    2. Arraste e solte a pasta com seu index.html (ou zip).
    3. Ele publica automaticamente e d um link tipo seu-nome.netlify.app.
  • Se quiser conectar GitHub depois, ainda mais automtico pra atualizaes.
  • SSL grtis, rpido no mundo todo, e super estvel.

3. Vercel (muito parecido com Netlify)

  • V para: https://vercel.com/new
  • Se no tiver GitHub, use o modo "Drag & Drop" ou crie uma conta rpida.
  • Arraste a pasta com o HTML publica em segundos.
  • Link: seu-projeto.vercel.app

4. GitHub Pages (clssico e 100% grtis)

  • Precisa de conta no GitHub (grtis).
  • Passos:
    1. Crie uma conta em github.com
    2. Crie um repositrio novo (ex: obrigado-page)
    3. Faa upload do index.html
    4. V em Settings Pages Selecione branch "main" e pasta "/root"
    5. Link: seuusuario.github.io/obrigado-page
